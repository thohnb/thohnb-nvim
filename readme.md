### Keybind

| Mode    | Phím          | Thao Tác                                                                                       |
|---------|---------------|------------------------------------------------------------------------------------------------|
| n  | `w`           | Di chuyển lên (↑)                                                                              |
| n  | `a`           | Di chuyển sang trái (←)                                                                        |
| n  | `s`           | Di chuyển xuống (↓)                                                                            |
| n  | `d`           | Di chuyển sang phải (→)                                                                        |
| n  | `sl`          | Mở cửa sổ dọc mới và chuyển đến thư mục chứa tập tin hiện tại                               |
| n  | `sk`          | Mở cửa sổ ngang mới và chuyển đến thư mục chứa tập tin hiện tại                              |
| n  | `<C-n>`       | Mở tài liệu mới                                                                                |
| n  | `<C-o>`       | Mở tệp trong cùng thư mục với tệp hiện tại sử dụng cửa sổ dấu nháy chính                       |
| n  | `<C-s>`       | Lưu tập tin hiện tại                                                                           |
| n  | `mw`          | Di chuyển cửa sổ lên trên                                                                      |
| n  | `md`          | Di chuyển cửa sổ sang phải                                                                     |
| n  | `ms`          | Di chuyển cửa sổ xuống dưới                                                                    |
| n  | `ma`          | Di chuyển cửa sổ sang trái                                                                     |
| n  | `@`           | Dùng Telescope với file-browser extension |
| n  | `<leader>cp`  | Chuyển đổi xem trước Markdown                                                                  |
| n  | `:`           | Mở dòng lệnh tìm kiếm fine                                                                     |
| Terminal| `<Esc>`       | Thoát khỏi focus khi sử dụng terminal (Terminal)                                                            |
| n  | `te`          | Tạo một tab mới                                                                                |
| n  | `<Tab>`       | Chuyển đến tab tiếp theo                                                                       |
| n  | `<S-Tab>`     | Chuyển đến tab trước đó                                                                        |
| n  | `<C-w>`       | Đóng tab hiện tại                                                                              |
| n  | `<C-f>`       | Tìm kiếm trực tiếp trong tệp với Telescope                                                     |
| n  | `<C-o>`       | Chuyển đến thư mục gốc của dự án hiện tại                                                     |
| n  | `pa`          | Thêm thư mục hiện tại vào danh sách dự án                                                      |
